# FraudGuard AI Pro

AI-powered real-time fraud detection and risk monitoring platform built using React, FastAPI, SQLite, WebSockets, and Scikit-learn.

---

## 🚀 Overview

FraudGuard AI Pro is a full-stack fraud detection system designed to identify suspicious financial transactions in real time.

The platform analyzes transaction activity, generates fraud risk scores, streams live alerts, and enables admin monitoring through an interactive dashboard.

---

## ✨ Features

- Real-time fraud detection
- Fraud risk scoring
- Live transaction monitoring
- Explainable fraud insights
- WebSocket live alerts
- Admin dashboard
- Transaction review and blocking
- Authentication system
- Dashboard analytics
- CSV export functionality
